# Netflix Data Analysis Dashboard

## Project Overview
This project focuses on analyzing Netflix content data to understand
trends in movies and TV shows across different countries, years,
and content ratings. The analysis is presented through an interactive
Power BI dashboard.

The objective of this project is to practice data cleaning,
data analysis, and dashboard creation using real-world data.

---

## Dataset
- The raw dataset is stored in the `data/raw` folder
- The cleaned dataset is stored in the `data/cleaned` folder
- The dataset contains information such as:
  - Content type (Movie / TV Show)
  - Title
  - Country
  - Release year
  - Rating
  - Duration
  - Date added to Netflix

---

## Data Cleaning
Data cleaning was performed before creating the dashboard.
The following steps were carried out:
- Removed duplicate records
- Handled missing and null values
- Standardized date formats
- Cleaned country and rating columns
- Created calculated fields for analysis

---

## Dashboard
- The dashboard was created using **Power BI**
- Interactive filters are available for:
  - Year
  - Country
  - Content Type
  - Rating
- Visuals included in the dashboard:
  - Movies vs TV Shows distribution
  - Content added over the years
  - Country-wise content analysis
  - Rating-wise content distribution

---

## Key Insights
- Movies dominate the Netflix content library
- Netflix content growth increased significantly after 2016
- The United States and India are among the top content-producing countries
- TV shows have shown strong growth in recent years
